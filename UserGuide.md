# User Guide

## Sign Up

1. Open the system.
2. Click **Sign Up**.
3. Enter name, phone, email, and password.
4. Submit the form.
5. The new account will be created as a normal user.

## Login

1. Click **Login**.
2. Enter email and password.
3. Normal users will be sent to the rental page.

Default normal user:

```text
Email: user@lzw.local
Password: user123
```

## Rent a Car

1. Login as a user.
2. Click **Rentals**.
3. Choose an available car.
4. Select pickup and return dates.
5. Enter purpose if needed.
6. Submit the rental order.

The system blocks double booking. If the same car already has an active rental during the selected date range, the order will not be created.

## Manage My Rental Orders

Users can see only their own rental orders. Users can edit or cancel their own orders only when the order status is `reserved`.
